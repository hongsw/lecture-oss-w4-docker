# Docker 입문 강의 개요

## 강의 목표
이 강의는 Docker의 기본 개념부터 실전 활용까지 체계적으로 학습할 수 있도록 설계되었습니다. 컨테이너 기술의 이해와 Docker를 활용한 애플리케이션 개발, 배포, 운영 방법을 습득하는 것을 목표로 합니다.

## 대상 수강생
- 컨테이너 기술에 관심 있는 개발자
- 개발 환경을 표준화하고 싶은 팀
- DevOps 환경을 구축하고자 하는 엔지니어
- 클라우드 네이티브 애플리케이션 개발자

## 필수 사전 지식
- 기본 리눅스 명령어 이해
- 프로그래밍 언어 최소 1개 이상 숙지
- 웹 개발 기초 지식
